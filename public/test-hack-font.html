Common Programming Ligatures

Arrow: -> →

Double arrow: => ⇒

Not equal: != ≠

Greater than or equal: >= ≥

Less than or equal: <= ≤

Triple equal: === ≡

Fat arrow: => ⇒

Lambda: => λ
